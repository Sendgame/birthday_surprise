Birthday Wish Link Website

How to use:
1. Open index.html in your browser.
2. Edit the birthday text in index.html.
3. Replace photo boxes with real images if you want.
4. Add your own music file named music.mp3 in this same folder.
5. Upload this folder to Netlify, Vercel, or GitHub Pages.
6. Share the generated link with the birthday person.

For photos:
- Add images like photo1.jpg, photo2.jpg
- Replace the <div class="photo">Add Photo 1</div> with:
  <img src="photo1.jpg" class="real-photo">
